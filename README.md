# LootFloat 🌊

LootFloat is a cozy quality-of-life mod for **Valheim** that helps dropped items stay visible and recoverable by making many of them float on water.

No more watching your loot quietly disappear into the depths. 🧌

## Features

* Makes supported dropped items float on water
* Helps keep loot visible and recoverable
* Works in singleplayer
* Tested in modded co-op
* Lightweight and focused
* Does not change recipes, inventories or item stats

## Installation

The recommended way to install LootFloat is through **r2modman** or the Thunderstore Mod Manager.

➡️ [Download LootFloat on Thunderstore](https://thunderstore.io/c/valheim/p/GoblinStudio/LootFloat/)

## Requirements

* Valheim
* BepInExPack for Valheim

Required dependencies are handled automatically when installing through a compatible mod manager.
